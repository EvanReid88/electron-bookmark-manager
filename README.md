# AngularElectron

Build a native desktop Angular Bookmark Manager App

- Angular v4.2
- Electron v1.7
- Angular CLI v1.4 

## Basic Usage

```shell
git clone
cd angular-electron
npm install

# build the app
npm run electron-build
```

to package app

electron-packager . testapp2 --platform=win32 --icon=src/assets/logo2.icns
